**Atlas** is an intelligent AI-powered project management system that:
- Generates complete project plans through conversational AI
- Automates task assignment and workflow management
- Provides real-time team collaboration
- Proactively identifies and manages project risks

- **Infrastructure:** Docker Compose (local development)
